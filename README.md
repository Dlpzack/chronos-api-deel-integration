## Environment Variables

The following environment variables are required for the application to run:

| Variable | Description | Required |
|----------|-------------|----------|
| DEEL_ACCESS_TOKEN | Access token for Deel API authentication | Yes |
| DEEL_API_URL | Base URL for Deel API (defaults to 'https://api.deel.com/v1') | No |
| DEEL_WEBHOOK_SIGNING_KEY | Signing key for validating Deel webhooks | Yes |

## Project setup

```bash
pnpm install:deps
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
